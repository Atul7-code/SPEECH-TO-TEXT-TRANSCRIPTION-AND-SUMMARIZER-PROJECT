# Audio Transcriber Pro

A powerful audio transcription and summarization tool built with Gradio and Vosk.

## Features
- High-accuracy speech-to-text transcription
- AI-powered text summarization
- Support for various audio formats
- User-friendly web interface

## How to Use
1. Upload an audio file or record directly
2. Click "Transcribe" to convert speech to text
3. Toggle "Generate Summary" for a concise summary
4. Download the transcription or summary as needed

## Running Locally
```bash
# Clone the repository
git clone https://huggingface.co/spaces/your-username/audio-transcriber
cd audio-transcriber

# Install dependencies
pip install -r requirements.txt

# Run the app
python app.py
```

## Deployment on Hugging Face Spaces
This app is designed to run on Hugging Face Spaces. The model will be automatically downloaded when the Space is created.
